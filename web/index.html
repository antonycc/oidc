Welcome to the OIDC Provider

This repository provides a comprehensive OIDC (OpenID Connect) Provider implementation with two authentication methods:

Getting Started

Click Login to choose your preferred authentication method.

Authentication Options

  • Direct OIDC Login: Exercise the OIDC Provider directly with a login form
  • Cognito Hosted UI: Use AWS Cognito's managed authentication that federates to this provider

For Developers

This implementation demonstrates:

  • PKCE (Proof Key for Code Exchange) flow
  • JWT token handling and validation
  • AWS Cognito federation
  • Dynamic configuration deployment
  • Comprehensive testing with Playwright and K6
